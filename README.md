# AutoJoinWG

Installation & Usage Guide
What is AutoJoinWG?
AutoJoinWG is a lightweight utility for World of Warcraft (3.3.5a) designed for players who want to automate the Wintergrasp queue process. It provides a simple on-screen toggle to enable or disable the auto-accept feature.

Installation
Navigate to your World of Warcraft directory.

Open the Interface folder, then the AddOns folder.

Create a new folder named AutoJoinWG.

Place the AutoJoinWG.toc and AutoJoinWG.lua files inside that folder.

Features
One-Click Toggle: Click the on-screen button to switch between ON (Green) and OFF (Red).

ON: Automatically accepts Wintergrasp invites and hides the popup.

OFF: Does nothing, allowing the standard Blizzard invite window to appear.

Draggable UI: Hold the [Shift] key and click-drag the button to move it anywhere on your screen.

Screen Clamping: The button cannot be dragged off the edge of your screen.

Slash Commands
If the button is lost or you want to center it perfectly, use the following command in the game chat:

/wgreset — Snaps the toggle button back to the center of the screen.

Troubleshooting
Addon not showing up? Ensure the folder name is exactly AutoJoinWG and that the file extensions are .lua and .toc (not .lua.txt).

Button hidden? Type /wgreset in chat to bring it back to the center.
